This is the schematic for my circuit. After using Ohm's Law to calculate the resistance, I determined that it needed to be 160 ohms. I rounded up to a 220-ohm resistor as it was the closest available value. The red, yellow, and green LED lights all have the same voltage drops, so I decided to use 220-ohm resistors for each LED.



This is my circuit. Each LED is connected to its own pin on the Arduino (pins 11–13). This setup allows direct input control for each LED from the Arduino. I chose the red, yellow, and green LED as they all contain the same voltage drop. Below is the code used in my Arduino to program the LED to blink and turn off.


void setup() {

// Pins 11, 12, and 13 are used as outputs

// It starts running after it's powered to the board

pinMode(11, OUTPUT);

pinMode(12, OUTPUT);

pinMode(13, OUTPUT);

}


void loop() {

// Starts on pin 11

// turn the red LED on (HIGH is the voltage level)

digitalWrite(11, HIGH);

// wait for a second

delay(1000);

// turns the LED off by making the voltage LOW

digitalWrite(11, LOW);

// wait for a second

delay(1000);

// turns the yellow LED on by making the voltage high

digitalWrite(12, HIGH);

// Waits for a second

delay(1000);

// turns off the LED by making the voltage low

digitalWrite(12, LOW);

delay(1000);

// turns on the green LED by making the voltage high

digitalWrite(13, HIGH);

// waits for a second

delay(1000);

// turns off the LED by making the voltage low

digitalWrite(13, LOW);

delay(1000);

}

If we are using 12 pins, we would need to use 20 mA or less. The calculations would be 20 mA X 12 = 240 mA. When running the 12 pins the current should not exceed 240 mA.
